# BMI Calculator

bhvr.dev (Bun + Vite + React + TypeScript) ベースの BMI 計算アプリです。

## 必須環境

- Bun 1.1+

## セットアップ

```bash
bun install
```

## 開発コマンド

```bash
bun run dev
bun run build
bun run preview
bun run test
bun run type-check
bun run lint
bun run format
```
