# BMI計算 要件定義書（軽量版）

## 概要

Webブラウザで動作するBMI計算アプリケーション。身長と体重を入力すると、BMI値、標準体重、標準体重との差を計算して表示します。ローカル環境で動作し、計算履歴を LocalStorage に保存します。

### 必須機能（Must Have）

- **REQ-001**: システムは身長（cm）と体重（kg）を数値入力フィールドで受け付けなければならない 🔵 *requirements.md + ユーザヒアリング2026-06-01より*

- **REQ-002**: システムはBMIの計算式（BMI = 体重kg ÷ (身長m)²）をトグルボタンで表示/非表示できなければならない 🔵 *requirements.md + ユーザヒアリング2026-06-01より*

- **REQ-003**: 身長と体重が入力された場合、システムはBMI値を小数点第1位まで計算して表示しなければならない 🔵 *requirements.mdより*

- **REQ-004**: システムはBMIの数値基準（低体重/普通体重/肥満等）を解説テキストとして表示しなければならない 🔵 *requirements.mdより*

- **REQ-005**: 身長が入力された場合、システムは標準体重（身長m² × 22）を計算して表示しなければならない 🔵 *requirements.md追加要件より*

- **REQ-006**: 体重と標準体重が計算された場合、システムは標準体重との差（体重 - 標準体重）を表示しなければならない 🔵 *requirements.md追加要件より*

- **REQ-007**: システムは計算結果（身長、体重、BMI、日時）をLocalStorageに保存しなければならない 🔵 *ユーザヒアリング2026-06-01より*

### 基本的な制約

- **REQ-401**: システムはローカル環境（PCブラウザ）でのみ動作しなければならない 🔵 *requirements.md前提条件より*

- **REQ-402**: システムはサーバーとの通信を行ってはならない（全てクライアントサイドで完結） 🔵 *tech-stack.md + requirements.mdより*

## 簡易ユーザーストーリー

### ストーリー1: BMI計算

**私は** 健康管理をしたいユーザー **として**
**身長と体重を入力してBMIを確認したい**
**そうすることで** 自分の体重が適正かどうかを判断できる

**関連要件**: REQ-001, REQ-003, REQ-004

**シナリオ**:
1. ユーザーが身長（例: 170cm）を入力
2. ユーザーが体重（例: 65kg）を入力
3. システムがBMI（22.5）を自動計算して表示
4. システムがBMI基準（普通体重）を表示

---

### ストーリー2: 標準体重の確認

**私は** ダイエット中のユーザー **として**
**自分の標準体重と現在の体重の差を知りたい**
**そうすることで** 目標体重を設定できる

**関連要件**: REQ-005, REQ-006

**シナリオ**:
1. ユーザーが身長（例: 170cm）を入力
2. システムが標準体重（63.6kg）を自動計算
3. ユーザーが体重（例: 65kg）を入力
4. システムが標準体重との差（+1.4kg）を表示

---

### ストーリー3: 計算式の理解

**私は** BMIの仕組みを知りたいユーザー **として**
**計算式を確認したい**
**そうすることで** BMIの計算方法を理解できる

**関連要件**: REQ-002

**シナリオ**:
1. ユーザーが「計算式を表示」ボタンをクリック
2. システムが計算式「BMI = 体重(kg) ÷ (身長(m) × 身長(m))」を表示
3. ユーザーが再度ボタンをクリックすると非表示になる

---

### ストーリー4: 計算履歴の保存

**私は** 定期的に体重管理するユーザー **として**
**過去の計算結果を確認したい**
**そうすることで** 体重の変化を追跡できる

**関連要件**: REQ-007

**シナリオ**:
1. ユーザーがBMI計算を実行
2. システムが計算結果を LocalStorage に自動保存
3. ユーザーがページを再読み込みしても履歴が残っている

## 基本的な受け入れ基準

### REQ-001: 身長・体重の入力

**Given（前提条件）**: アプリケーションが表示されている
**When（実行条件）**: ユーザーが身長と体重の入力フィールドに数値を入力する
**Then（期待結果）**: 
- 数値のみが入力可能
- 身長の範囲: 100-250cm
- 体重の範囲: 20-300kg

**テストケース**:
- [ ] **正常系**: 身長170cm、体重65kgを入力すると受け付けられる
- [ ] **主要な異常系**: 身長に文字列「abc」を入力するとエラーメッセージが表示される
- [ ] **境界値**: 身長99cm（範囲外）を入力するとエラーメッセージが表示される

---

### REQ-003: BMI計算と表示

**Given（前提条件）**: 身長と体重が正しく入力されている
**When（実行条件）**: 入力が完了する
**Then（期待結果）**: 
- BMI値が自動計算される
- 小数点第1位まで表示される（例: 22.5）
- BMI基準の判定が表示される（例: 「普通体重」）

**テストケース**:
- [ ] **正常系**: 身長170cm、体重65kg → BMI 22.5 が表示される
- [ ] **主要な異常系**: 身長が未入力の場合、BMIは計算されない

---

### REQ-005, REQ-006: 標準体重の計算と差分表示

**Given（前提条件）**: 身長が入力されている
**When（実行条件）**: 身長入力が完了する
**Then（期待結果）**: 
- 標準体重が自動計算される
- 体重が入力されている場合、標準体重との差が表示される
- 差はプラス/マイナス表示（例: +1.4kg または -2.3kg）

**テストケース**:
- [ ] **正常系**: 身長170cm → 標準体重 63.6kg が表示される
- [ ] **正常系**: 身長170cm、体重65kg → 標準体重との差 +1.4kg が表示される

---

### REQ-007: 計算履歴の保存

**Given（前提条件）**: BMI計算が実行された
**When（実行条件）**: 計算結果が表示される
**Then（期待結果）**: 
- 計算結果が LocalStorage に保存される
- 保存データ: 身長、体重、BMI、計算日時
- 最大10件まで保存（古いものから削除）

**テストケース**:
- [ ] **正常系**: 計算実行後、LocalStorage に履歴が保存される
- [ ] **主要な異常系**: LocalStorage が無効な環境では履歴保存をスキップする

## BMI基準値（参考）

| BMI | 判定 |
|-----|------|
| 18.5未満 | 低体重 |
| 18.5以上25未満 | 普通体重 |
| 25以上30未満 | 肥満(1度) |
| 30以上35未満 | 肥満(2度) |
| 35以上40未満 | 肥満(3度) |
| 40以上 | 肥満(4度) |

## 計算式

### BMI
```
BMI = 体重(kg) ÷ (身長(m) × 身長(m))
```

### 標準体重
```
標準体重(kg) = 身長(m) × 身長(m) × 22
```

### 標準体重との差
```
差(kg) = 現在の体重(kg) - 標準体重(kg)
```
