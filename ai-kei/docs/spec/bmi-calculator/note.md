## BMI計算ロジック

### 計算式
```
BMI = 体重(kg) ÷ (身長(m) × 身長(m))
```

### 標準体重の計算
```
標準体重(kg) = (身長(m) × 身長(m)) × 22
```

### BMI基準値
- 18.5未満: 低体重
- 18.5以上25未満: 普通体重
- 25以上30未満: 肥満(1度)
- 30以上35未満: 肥満(2度)
- 35以上40未満: 肥満(3度)
- 40以上: 肥満(4度)
